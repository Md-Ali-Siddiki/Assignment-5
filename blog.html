Javascript-DOM

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects, which can be manipulated with programming languages like JavaScript.

2. How do you select an element from the DOM?

To select an element from the DOM, use methods like document.getElementById() for IDs, document.querySelector() for any CSS selector, or document.getElementsByClassName() for class names.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is when one event listener is added to a parent element to handle events for its child elements. This reduces the number of event listeners needed and makes the code simpler.

4. How do you manipulate an element's attributes and styles using the DOM?

To change an element's attributes and styles, use element.setAttribute() for attributes and element.style for styles. For example, element.setAttribute('src', 'image.jpg') changes an image's source, and element.style.color = 'red' changes the text color to red.
